
### 2.Доступные страницы
1. HOME ("/"): рекламный слайдер, отображение туров, статей
2. ABOUT ("/about"): описание вебсайта и компании 
3. TOURS ("/tours"): отображение списка туров. Можно перейти на каждый тур и посмотреть подробнее
4. TOUR_DETAILS ("/tours/:id"): Подробности о туре
5. CART ("/cart"): Стаб страницы корзины - статичная для примера
6. CONTACT ("/contact"): Контакты компании 
7. ARTICLES ("/articles"): Список статей (постов) от компании 
8. ARTICLES_SLUG ("/articles/:slug"): Страница для просмотрп статьи 
9. PROFILE ("/profile"): страница с информацией о зарегестрированном пользователе пользователе и его фотографиями  
11. PROFILE_EDIT ("/profile/edit"): редактирование информации о пользователе
12. DASHBOARD ("/dashboard"): личный кабинет пользователя с его данными (личные, банковсие, его туры)
13. NOTIFICATIONS ("/notifications"): уведомления для пользователя (появляются, например,  при добавлении банковских данных)
14. MY_PHOTOS: ("/my-photos"): страниц пользователя для загрузки\скачки фотографий
15. ADMIN ("/admin"): входная страница админка 
16. ADMIN_CREATE_TOUR ("/admin/tours"): страница создания тура 
17. ADMIN_EDIT_TOUR ("/admin/tours/:id"): редактирование конкретного тура - часть 1 
18. ADMIN_TOUR_CONTINUE ("/admin/tours/:id/continue"): редактирование дополнительной информации  о туре (фото, теги и пр)
19. ADMIN_USERS ("/admin/users"): список зарегестрированных пользователей 
20. ADMIN_USER_GALLERY ("/admin/users/:id/photos"): фотки конкретного польозвателя
21. NOT_FOUND ("/*"): not found page

### 4. Внешние интеграции
1. ZOHO (CRM) - отправляем в ZOHO всех зарегестрированных пользователей
2. CLAUDINARY (storage service) - сервис для хранения статики, испольуем для хранения файлов
